Force-regenerate `~/.claude/home_index.md` by running `~/.bin/refresh-home-index.sh`.

Use this after creating, renaming, or deleting a project directory mid-session — otherwise the daily auto-refresh on `/start-session` is enough.

Steps:

1. Run `bash ~/.bin/refresh-home-index.sh`
2. Read the new `~/.claude/home_index.md` and confirm the change is reflected
3. Report what changed (one line — added/removed entries, or "no structural change")
